## Usage Guidelines

1. When citing a source in a skill, reference by ID (e.g., "See S-001")
2. If a source contradicts another, prefer the higher-numbered primary source
3. Community sources (S-008) require cross-verification against primary sources
4. Always note the date of last verification for time-sensitive sources
